# 1219co_suimei

シンプルなフロントエンドのみで動作する四柱推命チャート計算ツールです。仕様書に記載されたアルゴリズムを vanilla JavaScript で実装しています。

## 使い方

1. 任意の静的サーバーで `index.html` を開く（例: `python -m http.server`）。
2. 生年月日、出生時刻、性別、日付切り替えルールを入力して「命式を計算」をクリック。
3. 年柱〜時柱の干支、通変星、十二運、蔵干、身強/身弱判定を確認できます。

## 実装メモ

- 年柱は立春（毎年 2 月 4 日目安）で切り替え。
- 月柱は仕様書の節気代表日を用いて月支を決定。
- 日柱はユリウス日計算、時柱は日上起時法（0 時切り替え／23 時切り替えを選択可）。
- 通変星・十二運は日干を基準に算出。大運の方向も表示します。
